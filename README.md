# Conversation Quality Self‑Evaluator Agent

## Overview
This project focuses on **proactively monitoring conversation health** in AI‑driven CRM interactions. It addresses silent failure modes such as repetition, stagnation, and lack of progress that often lead to customer churn without explicit complaints.

The agent treats conversation quality as a **control signal**, not a post‑hoc metric, enabling early adaptation or escalation.

---

## Key Features
- Repetition detection
- Stagnation and progress tracking
- Quality scoring independent of sentiment
- Adaptive policy triggers
- Proactive escalation support
